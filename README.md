# primeros_pasos_git
Aprender fundamentos de git

## Como guardar cada cambio del master(main)
preparar los cambios para ser publicados al repositorio remoto
* git add <nombre_archivo.php>
* git commit -m "escribimos descripcion de nuestro commit"
* git push origin main => esto para publicar los cambios en el remoto

## Creación de ramas en git

* para crear la rama => git checkout -b rama1
* para preparar la rama => git commit -m "primera rama"
* para publicar => git push origin rama1
